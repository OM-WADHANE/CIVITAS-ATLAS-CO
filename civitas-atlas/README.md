# Civitas Atlas — Landing Website

Static landing website for Civitas Atlas Technologies.

**Stack:** HTML5 · CSS3 · Vanilla JavaScript  
**Deploy:** Open `index.html` directly, or deploy any static host (Vercel, Netlify, GitHub Pages).

## Files

```
index.html          — full single-page site
style.css           — all styles
script.js           — all interactivity
assets/
  images/
    mahaatlas-brand.png   — MahaAtlas product banner
    founder.jpg           — Founder photo
vercel.json         — Vercel static deployment config
```

## Deploy on Vercel

1. Push this folder to a GitHub repo
2. Import the repo on vercel.com
3. Set **Root Directory** to the folder containing `index.html`
4. **Framework Preset** → select **Other** (not Next.js)
5. Leave Build Command and Output Directory blank
6. Deploy

## Deploy on Netlify

Drag and drop this entire folder onto netlify.com/drop — done.
